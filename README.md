# AWS GenAI Pro Exam Portal (AIP-C01)

Interactive web application for preparing for the AWS Generative AI Professional exam (AIP-C01).

## 🚀 Features
- Study Mode: View structured lessons and mind maps
- Flashcards: Interactive Q&A with progress tracking
- Mock Exams: Timed, randomized questions
- Scenario Mode: Architecture & RAG pipeline challenges
- Progress Dashboard: Track weak domains

## 🧩 Tech Stack
- React / Vite (or Next.js)
- Tailwind CSS
- Local JSON content (or fetch from materials repo)
- GitHub Pages / Netlify / Vercel deployment
